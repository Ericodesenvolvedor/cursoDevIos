Nome da cor

No CSS, as cores podem ser nomeadas pelo nome (em Inglês), como White para branco, green, para verde, blue, para azul e assim por diante.

Com hexadecimal

Os valores hexadecimais são, na verdade, apenas uma maneira diferente de representar valores RGB. Em vez de usar três números entre 0 e 255, você usa seis números hexadecimais.

Rgb

A notação funcional rgb() expressa uma cor de acordo com seus componentes vermelho, verde e azul. Vale resaltar que os valores começam no 0 e vão até 255

Hsl

HSL e HSV são representações alternativas do modelo de cores RGB, projetado na década de 1970 por pesquisadores de computação gráfica para se alinhar mais de perto com a forma como a visão humana percebe os atributos de criação de cores.
