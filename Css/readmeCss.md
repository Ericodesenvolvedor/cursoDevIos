# Introdução ao Css

- Criada em 1995
- Destaque entre 1997 e 1999

## Precedência no Css

- Inline: o estilo é colocado dentro da propria tag
- Incorporado: O estilo vai dentro da tag head (style)
- Externo: é criado um arquivo separadamente para ele.
